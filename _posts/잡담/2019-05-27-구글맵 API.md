## 구글 API기 받기

> https://developers.google.com/maps/documentation/javascript/get-api-key


카드등록하고 설명대로 진행하면 쉽게 가져올 수 있다.  

1년간 무료이며 요청횟수에 따라 조기종료될 수 있다.  

## 마커가 있는 구글지도 추가

https://developers.google.com/maps/documentation/javascript/adding-a-google-map

```js
<script async defer
  src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap">
</script>
```
`YOUR_API_KEY` 부분에 받아온 키를 삽입하면 된다.  

```html
<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<script
	src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<meta charset="UTF-8">
<title>JSP / Servelet Class - kouzie(2019. 5. 28.-오전 9:05:39)</title>
<script>
	$(document).ready(function() {

	});
</script>
<style>
#map {
	height: 400px;
	width: 100%;
}
</style>
</head>
<body>
	<h3>My Google Maps Demo</h3>
	<!--The div element for the map -->
	<div id="map"></div>
	<script>
    //아래의 구글맵 스크립트를 로드되면 initMap()함수를 콜백하는데
    //중심좌표, 마커 등을 설정해서 지동에 표시할 수 있다.  
		function initMap() { 
			var uluru = {
				lat : -25.344,
				lng : 131.036
      };


      //맵객체 첫번째 인자로 맵을 띄울 div태그, 
      //두번째 인자로 위에서 설정한 줌인자와 좌표객체를 넣는다.
			var map = new google.maps.Map(
				document.getElementById('map'), 
				{
					zoom : 5,
					center : uluru
				}
			);
      
      //마커를 찍고싶을땐 좌표객체와 찍을 맵객체를 넣어준다.  
			var marker = new google.maps.Marker({
				position : uluru,
				map : map
			});
		}
	</script>
	<script async defer
    src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap">
  </script>
</body>
</html>
```
